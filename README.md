# CATS CRADLE — Biometric String Art

Interactive futuristic hand-tracking artwork built with MediaPipe Hands and HTML5 Canvas.

Users weave glowing neon strings between their fingertips in real time using webcam-based hand tracking.

---

## Features

- Real-time hand tracking
- Glowing elastic string simulation
- Cyberpunk HUD interface
- Finger landmark visualization
- Dynamic particle trails
- Responsive fullscreen canvas
- Zero dependencies except CDN imports

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- MediaPipe Hands
- Canvas API

---

## How To Run

Simply open:

```txt
index.html
```

in a browser.

OR run using VS Code Live Server.

---

## Controls

| Action | Result |
|---|---|
| Show two hands | Activate string art |
| Move fingers | Dynamic string movement |
| Stretch hands apart | Elastic tension effect |
| Remove hands | Tracking pauses |
